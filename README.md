# QA Railway Automation Testing Project

### Tech Stack
* **Language:** Java
* **Framework:** Selenium WebDriver
* **Test Runner:** TestNG
* **Pattern:** Page Object Model (POM)
* **Data Management:** Data Objects & Enums

---

## Test Case Details

| Suite | ID | Description | Steps | Expected Behavior |
| :--- | :--- | :--- | :--- | :--- |
| **Login** | 1 | User can log into Railway with valid username and password | 1. Navigate to QA Railway Website<br>2. Click on "Login" tab<br>3. Enter valid Email and Password<br>4. Click on "Login" button | User is logged into Railway. Welcome user message is displayed. |
| **Login** | 2 | User cannot login with blank "Username" textbox | 1. Navigate to QA Railway Website<br>2. Click on "Login" tab<br>3. User doesn't type any words into "Username" textbox but enter valid information into "Password" textbox<br>4. Click on "Login" button | User can't login and message "There was a problem with your login and/or errors exist in your form. " appears. |
| **Login** | 3 | User cannot log into Railway with invalid password | 1. Navigate to QA Railway Website<br>2. Click on "Login" tab<br>3. Enter valid Email and invalid Password<br>4. Click on "Login" button | Error message "There was a problem with your login and/or errors exist in your form." is displayed |
| **Login** | 4 | System shows message when user enters wrong password many times | 1. Navigate to QA Railway Website<br>2. Click on "Login" tab<br>3. Enter valid information into "Username" textbox except "Password" textbox.<br>4. Click on "Login" button ("Invalid username or password. Please try again" is shown)<br>5. Repeat step 3 and 4 three more times. | User can't login and message "You have used 4 out of 5 login attempts. After all 5 have been used, you will be unable to login for 15 minutes." appears. |
| **Login** | 5 | User can't login with an account hasn't been activated | Pre-condition: a not-active account is existing<br>1. Navigate to QA Railway Website<br>2. Click on "Login" tab<br>3. Enter username and password of account hasn't been activated.<br>4. Click on "Login" button | User can't login and message "Invalid username or password. Please try again." appears. |
| **Logout** | 6 | User is redirected to Home page after logging out | 1. Navigate to QA Railway Website<br>2. Login with valid Email and Password<br>3. Click on "FAQ" tab<br>4. Click on "Log out" tab | Home page displays. "Log out" tab is disappeared. |
| **Create account** | 7 | User can't create account with an already in-use email | Pre-condition: an actived account is existing<br>1. Navigate to QA Railway Website<br>2. Click on "Register" button | Error message "This email address is already in use." displays above the form. |
| **Create account** | 8 | User can't create account while password and PID fields are empty | 1. Navigate to QA Railway Website<br>2. Click on "Register" tab<br>3. | **Create account** | 9 | User create and activate account | 1. Navigate to QA Railway Website (Home page is shown with guide containing href "create an account" to "Register" page)<br>2. Click on "Create an account" (Register page is shown)<br>3. Enter valid information into all fields<br>4. Click on "Register" button ("Thank you for registering your account" is shown)<br>5. Get email information (webmail address, mailbox and password) and navigate to that webmail<br>6. Login to the mailbox<br>7. Open email with subject containing "Please confirm your account" and the email of the new account at step 3<br>8. Click on the activate link | Redirect to Railways page and message "Registration Confirmed! You can now log in to the site" is shown |
| **Reset password** | 10 | Reset password shows error if the new password is same as current | Pre-condition: an actived account is existing<br>1. Navigate to QA Railway Login page<br>2. Click on "Forgot Password page" link<br>3. Enter the email address of the activated account<br>4. Click on "Send Instructions" button<br>5. Login to the mailbox (the same mailbox when creating account)<br>6. Open email with subject contaning "Please reset your password" and the email of the account at step 3<br>7. Click on reset link (Form "Password Change Form" is shown with the reset password token)<br>8. Input same password into 2 fields "new password" and "confirm password"<br>9. Click Reset Password | Message "The new password cannot be the same with the current password" is shown |
| **Reset password** | 11 | Reset password shows error if the new password and confirm password doesn't match | Pre-condition: an actived account is existing<br>1. Navigate to QA Railway Login page<br>2. Click on "Forgot Password page" link<br>3. Enter the email address of the activated account<br>4. Click on "Send Instructions" button<br>5. Login to the mailbox (the same mailbox when creating account)<br>6. Open email with subject contaning "Please reset your password" and the email of the account at step 3<br>7. Click on reset link (Form "Password Change Form" is shown with the reset password token)<br>8. Input different input into 2 fields "new password" and "confirm password"<br>9. Click Reset Password | Error message "Could not reset password. Please correct the errors and try again." displays above the form.<br><br>Error message "The password confirmation did not match the new password." displays next to the confirm password field. |
